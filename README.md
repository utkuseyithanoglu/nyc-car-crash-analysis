# 🚗 NYC Car Crash Analysis & Prediction

This project analyzes New York City car crash data to identify key risk factors and build predictive models for accident severity and patterns.

## 📊 Objectives
- Perform Exploratory Data Analysis (EDA) on NYC crash data  
- Identify high-risk boroughs, time periods, and contributing factors  
- Build predictive models for crash outcomes  
- Analyze time series patterns in accident frequency  

## 🧠 Methods & Models
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Linear Regression
- Random Forest + GridSearch
- ARIMA / SARIMA (time series forecasting)

## 🛠️ Tech Stack
- Python (pandas, numpy, scikit-learn, statsmodels)
- Data Visualization (matplotlib, seaborn)
- Jupyter Notebook

## 📁 Project Structure
- `eda_parts.ipynb` → Data exploration and visualization  
- `feature_engineering.ipynb` → Data preprocessing and feature creation  
- `linear_reg.ipynb` → Regression modeling  
- `rf_model.ipynb` → Random Forest and hyperparameter tuning  
- `arima.ipynb` / `sarima.ipynb` → Time series modeling  
- `time_series.ipynb` → Trend and seasonal analysis  

## 📈 Key Insights
- Crash frequency varies significantly by borough and time of day  
- Peak accident hours identified during high traffic periods  
- Certain features strongly influence crash outcomes  
- Time series models capture seasonal crash patterns
